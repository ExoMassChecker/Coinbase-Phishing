Option 1: A Record (Recommended)

  4. For www subdomain, create another A record with name "www"

  1. Create a CNAME record:
    • Name: @ or www
    • Type: CNAME
    • Value: Loading...

URL Redirection (Alternative)

If you can't modify DNS records, use URL forwarding:

  1. In your domain control panel, find "URL Forwarding" or "Domain Forwarding"
  2. Set up forwarding:
    • From: yourdomain.com
    • To: Loading...
    • Type: Masked/Frame redirect (to hide the real URL)

Propagation Time

DNS changes can take 24-48 hours to fully propagate worldwide. However, they usually work within a few minutes to a few hours.

Use the "Test Domain" button in your domain list to verify the setup is working.

SSL/HTTPS

For maximum authenticity, your domain should use HTTPS. The server automatically handles SSL certificates, but ensure your domain forwards to https:// not http://.

Important Notes

  • Legal Compliance: Only use domains you own and have permission to use
  • Testing: Always test your setup before deployment
  • Monitoring: Use the admin panel to monitor traffic and captures
  • Cleanup: Remember to remove DNS records when testing is complete

Management Dashboard User Guide

Getting Started

  1. Create Admin Accounts: Go to the Admin Management tab and create your first admin account. This will allow access to the monitoring dashboard.
  2. Link Domains: In the Domain Linking tab, add your domains to generate phishing URLs.
  3. Access Admin Panel: Click "Access Admin Panel" to open the monitoring interface where you can view captured data.

Admin Management

  • Create Accounts: Add new admin accounts with usernames and passwords
  • Test Login: Use the "Test Login" button to verify credentials work
  • View Status: Monitor which admins are active and when they last logged in
  • Delete Accounts: Remove admin accounts when no longer needed

Domain Management

  • Add Domains: Enter your domain names (e.g., secure-coinbase.com)
  • Generated URLs: Each domain creates multiple endpoint URLs automatically
  • Copy URLs: Use the copy button to get URLs for deployment
  • Test Domains: Verify domain functionality with the test button
  • SSL Status: Monitor SSL certificate status for each domain

Analytics & Monitoring

  • Live Statistics: View real-time counts of active admins, domains, and captures
  • Success Rate: Monitor overall operation effectiveness
  • Admin Panel: Access detailed session monitoring and data capture

Keyboard Shortcuts

  • Ctrl/Cmd + 1: Switch to Overview tab
  • Ctrl/Cmd + 2: Switch to Admin Management tab
  • Ctrl/Cmd + 3: Switch to Domain Linking tab
  • Ctrl/Cmd + 4: Switch to Analytics tab
  • Ctrl/Cmd + E: Export dashboard data

Pro Tips

  • Start with creating one admin account, then test login to verify it works
  • Use descriptive domain names that look legitimate (e.g., secure-coinbase-verify.com)
  • All data is saved locally in your browser - export regularly for backup
  • The admin panel provides real-time session monitoring and verification tools
  • Use the Test Login feature to verify admin credentials before going live

Advanced Features

  • Data Export: Export all dashboard data as JSON for backup/analysis
  • Clear Data: Reset dashboard to start fresh (use with caution)
  • Real-time Updates: Dashboard automatically syncs with backend when available
  • Session Management: Track admin login history and activity
